# Rubix Assets

This repository serves as the centralized, trusted source of truth for Rubix network configurations and assets. It is maintained by the Rubix core team.

Currently, this repository hosts the **FullNode Registry**, which provides a verified list of active Rubix FullNodes. Explorers, regular nodes, and external applications can query this registry to discover trusted nodes for network synchronization and data retrieval.

## Files

- `fullnodes.json`: The active registry of trusted FullNodes, categorized by network (mainnet/testnet).
